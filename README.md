ذا البرنامج هو مشغل قنوات IPTV احترافي بواجهة رسومية مكتوب بلغة بايثون باستخدام tkinter و VLC، واسمه:
👉 Show IPTV Pro MAX 🔥
سأعطيك وصفًا واضحًا ومختصرًا + احترافيًا يمكنك استخدامه:

🧠 وصف البرنامج
Show IPTV Pro MAX هو تطبيق سطح مكتب متكامل يتيح للمستخدم تشغيل ومشاهدة القنوات التلفزيونية عبر الإنترنت (IPTV) بسهولة واحترافية، مع واجهة حديثة وإمكانيات متقدمة لإدارة القنوات.

⚙️ وظيفة البرنامج
يقوم البرنامج بـ:


تشغيل قنوات IPTV من ملفات M3U / M3U8


أو من خلال رابط مباشر (URL)


عرض القنوات داخل مشغل فيديو مدمج باستخدام VLC



🎬 أهم المميزات
📺 تشغيل القنوات


تشغيل مباشر للقنوات عبر الإنترنت


دعم بروتوكولات متعددة (http / https / rtmp / rtp)


استخدام محرك VLC لضمان جودة واستقرار التشغيل



📂 استيراد القنوات


تحميل ملف M3U من الجهاز


تحميل قائمة قنوات من رابط (IPTV URL)



🔍 البحث والتصفية


البحث عن القنوات بالاسم


تصنيف القنوات حسب الفئة (رياضة، أفلام، أخبار…)


فلترة ذكية وسريعة



⭐ المفضلة


إضافة أو إزالة القنوات من المفضلة


حفظ المفضلة تلقائيًا في ملف data.json



🖼️ عرض الشعارات


تحميل وعرض شعار القناة تلقائيًا


يتم التحميل في الخلفية (بدون تجميد البرنامج)



🎬 الترجمة


دعم تحميل ملفات ترجمة خارجية (SRT)



🎛️ التحكم


تشغيل / إيقاف مؤقت


التحكم في الصوت


اختصارات لوحة المفاتيح:


Space → تشغيل/إيقاف


F → إضافة للمفضلة


Ctrl+O → فتح ملف


Ctrl+U → تحميل URL





🔲 وضع ملء الشاشة (تكبير)


تكبير الفيديو وإخفاء القائمة الجانبية


الرجوع بزر ESC أو نقر مزدوج



💾 حفظ البيانات


حفظ:


القنوات المفضلة


آخر قناة تم تشغيلها




في ملف JSON محلي



⚡ الأداء


استخدام Threads لتجنب التهنيج


تحديث واجهة المستخدم كل ثانية (عرض الوقت / الحالة)



🖥️ واجهة المستخدم


تصميم داكن احترافي 🎨


قائمة جانبية تحتوي على:


البحث


التصنيفات


قائمة القنوات




شاشة عرض فيديو كبيرة


شريط تحكم أسفل الفيديو



🧩 التقنيات المستخدمة


tkinter → واجهة المستخدم


vlc → تشغيل الفيديو


requests → تحميل البيانات والشعارات


PIL → معالجة الصور


threading → تحسين الأداء


json → حفظ البيانات



🎯 الهدف من البرنامج
إنشاء منصة IPTV احترافية على الكمبيوتر تجمع بين:


سهولة الاستخدام


سرعة الأداء


تنظيم القنوات


تجربة مشاهدة شبيهة بالتطبيقات المدفوعة
